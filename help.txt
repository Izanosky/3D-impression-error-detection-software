================================================
  AYUDA - Sistema de Monitorización 3D
================================================

== REQUISITOS PREVIOS ==
  - Python 3.10+
  - Node.js 18+ y npm
  - Base de datos en Firebase (para historial y usuarios)

== INSTALACIÓN ==

  1. Backend (Servidor/Raspberry Pi):
     cd backend
     python -m venv .venv
     
     # En Windows:
     .venv\Scripts\activate
     # En Linux/Mac:
     source .venv/bin/activate
     
     pip install -r requirements.txt

  2. Frontend (Cliente):
     cd frontend
     npm install

== CONFIGURACIÓN ==

  1. Backend (OctoPrint):
     La primera vez que ejecutes el backend, te pedirá por consola:
     - URL de OctoPrint (ej: http://localhost)
     - API Key de OctoPrint
     Esto se guardará automáticamente en un entorno local.

  2. Frontend (Firebase):
     Crea un archivo .env en la carpeta "frontend" con tus credenciales:
     VITE_FIREBASE_API_KEY=...
     VITE_FIREBASE_AUTH_DOMAIN=...
     VITE_FIREBASE_PROJECT_ID=...
     VITE_FIREBASE_STORAGE_BUCKET=...
     VITE_FIREBASE_MESSAGING_SENDER_ID=...
     VITE_FIREBASE_APP_ID=...

== EJECUCIÓN ==

  Terminal 1 - Backend:
    cd backend
    # En Windows: .venv\Scripts\activate
    # En Linux:   source .venv/bin/activate
    uvicorn app.main:app --host 0.0.0.0 --port 8000

  Terminal 2 - Frontend:
    cd frontend
    npm run dev

== MODELO IA (ONNX) ==

  La inferencia ahora se realiza directamente en el navegador del usuario
  (Frontend) usando ONNX Runtime Web, sin sobrecargar el servidor backend.
  
  Asegúrate de colocar tu modelo entrenado y exportado a formato ONNX en:
  frontend/public/model/best.onnx

== PROBLEMAS COMUNES (Linux / Raspberry Pi) ==

  1. "python3.10: command not found"
     → Instalar Python 3.10: sudo apt install python3.10 python3.10-venv

  2. "No se pudo crear el entorno virtual"
     → Falta el paquete venv: sudo apt install python3.10-venv

  3. No veo la carpeta .venv
     → Es oculta en Linux. Usa: ls -a

  4. npm no encontrado
     → Instalar Node.js: sudo apt install nodejs npm
     → O usar nvm: https://github.com/nvm-sh/nvm
